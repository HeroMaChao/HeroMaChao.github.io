打开CMD的方式

1、开始+系统+命令提示符

2、Win键+R 输入cmd打开控制台

3、在任意的文件夹下面，按shift键+鼠标右键点击，在此处打开命令行窗口

4、资源管理器的地址栏前面加上cmd路径

常用的Dos命令

常用基础命令

dir 查询所有文件

md 创建文件夹

小于号 （>）创建文件

rd 删除文件夹

cd 进入指定目录（change directory）

cd.. 退到根目录

cd\ 退回到根目录

del 删除文件

exit 退出dos命令行

cls 清理屏幕（clear screen）

操作命令

ipconfig 查询ip配置

ping 尝试网络是否能够使用

java -version 查看java版本

super注意点：

1、super调用父类的构造方法，必须在构造方法的第一个

2、super必须只能出现在子类的方法或者构造方法中

3、super和this不能同时调用构造方法
